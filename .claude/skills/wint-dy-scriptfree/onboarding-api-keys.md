# API Key 引导（Agent 必读 — Cursor / Claude Code / Codex 通用）

一键部署时 **Agent 必须主动交互**。不要丢给用户「自己去改配置文件」。

---

## 第一步：用大白话说明（setup.py 跑完后）

**禁止**堆术语（DashScope、Paraformer、ASR 等可以括号带过，正文要用普通人能懂的话）。参考下面话术，可略作调整：

---

**要给用户讲清楚这几件事：**

1. **这是干什么的**  
   有些抖音视频自带字幕，可以直接转成文字，**不用花钱、也不用配 Key**。  
   但有些视频**没有字幕**，就需要借助阿里云的「语音转文字」服务，把视频里人说的话识别出来。  
   配的这个 Key，就是用来调用这项服务的「通行证」。

2. **钱怎么算**  
   - 新注册阿里云账号，通常有**免费试用额度**（能转多少分钟以控制台显示为准，一般够试几十条短视频）  
   - 免费额度用完后，大约 **几分钱一分钟** 视频（例如 8 分钟视频大概 6 分钱）  
   - 具体价格以 [阿里云百炼计费说明](https://help.aliyun.com/zh/model-studio/billing) 为准；部署前可 **WebSearch** 查最新免费额度政策

3. **不配会怎样**  
   - ✅ 自带字幕的视频：照常能用  
   - ❌ 没有字幕的视频：转不出文字，会报错  
   - 口播稿润色（`/rewrite`）：**不受影响**，用的是你 Agent 的对话额度，不需要这个 Key

4. **Key 存哪里**  
   Agent 帮你写入本项目的 `config.local.yaml`，**不会上传到 GitHub**（已在 .gitignore 里）。你**不用**自己改文件、也不用敲 `export` 命令。

---

## 第二步：必须让用户二选一（三端统一体验）

**目标：** 用户明确选「现在配置」或「暂不配置」，**禁止**模糊跳过。

### 优先方式（Cursor 有 AskQuestion 工具时）

调用 **AskQuestion**，弹出选项：

| 字段 | 内容 |
|------|------|
| **问题** | 要不要现在配置阿里云语音转文字的 Key？（没有字幕的视频需要它） |
| **选项 1** | `configure_now` — **现在配置**（我按步骤教你拿 Key，你来粘贴） |
| **选项 2** | `skip_for_now` — **暂不配置**（只能处理自带字幕的视频；以后说「帮我配置阿里云百炼」即可） |

用户点选之前，先把第一步的大白话说明发在同一轮对话里。

### 回退方式（Claude Code / Codex / 无 AskQuestion 时）

在对话中**明确列出两个选项**，并**等待用户回复**（接受回复 `1`/`2`、`configure_now`/`skip_for_now`、或同义中文）：

```
请选一个（回复序号或文字即可）：
1️⃣ 现在配置 — 我一步步教你拿 Key，你粘贴给我
2️⃣ 暂不配置 — 先用有字幕的视频；以后再说「帮我配置阿里云百炼」
```

**禁止**不等待用户选择就继续；**禁止**默认替用户选「暂不配置」。

---

## 第三步 A：用户选「暂不配置」

- 简短确认：自带字幕的视频仍可用  
- 告知触发词：「帮我配置阿里云百炼 / DashScope / 语音 Key」  
- 继续部署：`python scripts/extract.py --check`

---

## 第三步 B：用户选「现在配置」

**禁止**只说一句「请把 Key 发我」。必须按顺序做：

### B1. 联网核对（推荐）

用 **WebSearch** 查「阿里云百炼 API Key 获取 2026」或打开官方文档，确认步骤无大改。官方入口：  
https://help.aliyun.com/zh/model-studio/get-api-key

### B2. 给用户可照着做的步骤（小白版）

1. **准备账号** — https://www.aliyun.com 注册/登录，**完成实名认证**  
2. **进入百炼控制台** — https://bailian.console.aliyun.com/ ，首次同意服务协议  
3. **开通模型服务** — 若提示未开通，点去开通 → 按量付费（有免费额度）  
4. **创建 API Key** — 设置 → API-KEY 管理 → 华北2（北京）→ 创建 → 权限全部 → 复制 `sk-` 开头密钥  
5. **粘贴给 Agent** — Agent 写入 `config.local.yaml` 的 `secrets.dashscope_api_key`  
6. **验证** — `python scripts/extract.py --check`

### B3. 用户粘贴 Key 后

Agent 直接写入配置并验证，回复「已配好，可以用了」。

---

## 延后配置触发词

用户之后说以下任一，重新走 **第二步 + 第三步 B**：

- 「帮我配置阿里云百炼」
- 「帮我配置 DashScope」
- 「配置语音 Key / ASR Key」
